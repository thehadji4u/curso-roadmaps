// princípio desta grade
As semanas 1–6 constroem a base (HTML/CSS + Git + Python fundamentos) sem dependências cruzadas. A partir da semana 7, JavaScript entra em paralelo com Python pois compartilham lógica de programação. Docker chega na semana 11 porque você já tem algo para containerizar. n8n começa na 13 aproveitando que você já sabe HTTP e APIs. Agentes de IA entram na 19 quando você já domina chamadas a APIs e lógica assíncrona. OpenClaw na 21 porque Playwright/automação de browser precisa de fundamentos web sólidos. As últimas semanas são de projetos integrados onde tudo se une.
