Por que isso importa?

O próximo nível da automação web

Automação tradicional com Playwright requer que você saiba exatamente qual seletor usar — button#submit, .checkout-btn. Mas e quando o site muda o HTML? E quando você precisa automatizar um site que nunca viu antes?

Com Vision LLMs (GPT-4o, Claude 3 Opus) você tira um screenshot da página, manda para o modelo e pergunta: "Onde está o botão de compra? Qual é o preço atual?". O modelo enxerga a página como um humano — sem precisar de seletores.

Combinando Playwright + LLM Vision + ReAct pattern, você cria agentes que navegam autonomamente — lêem, decidem, clicam, preenchem formulários, coletam dados — tudo controlado por linguagem natural.

🔥 Por que um engenheiro sênior usa isso hoje?
Times de QA usam Vision agents para testar UIs que mudam frequentemente sem precisar atualizar seletores. Times de dados usam para fazer scraping resiliente em sites com proteções anti-bot sofisticadas.

🤖 Analogia
Imagine dois tipos de robôs numa fábrica. O robô antigo precisa de instruções precisas: "Mova o braço 30cm à direita, desça 15cm, feche a garra". Se qualquer coisa mudar de posição, ele erra. O robô moderno tem câmera e IA: ele vê a peça, reconhece o que é, e decide como pegar — mesmo que ela esteja em posição diferente. Automação web com Vision LLM é a transição do robô "cego" para o robô que enxerga.

Conceito Central

Arquitetura: Vision + ReAct + Browser

A arquitetura combina três componentes: Playwright como executor de ações no browser, Vision LLM como "olhos" que interpretam screenshots, e o ReAct Loop como cérebro que decide qual ação tomar a seguir.

O que é Vision-Based Interaction

Em vez de usar seletores, o agente descreve o elemento em texto — "o botão verde de checkout" — e o Vision LLM localiza na screenshot atual. Isso torna a automação semanticamente robusta: funciona mesmo se o HTML mudar, desde que a aparência visual seja similar.

Seletor Tradicional

Frágil: quebra se a classe CSS mudar

await page.click(
  "#main-nav > ul > li:nth-child(3)"
)

Vision-Based

Robusto: entende o contexto visual

Vision Helper — Screenshot → Base64 → LLM
Função que tira screenshot e pede ao GPT-4o para analisar
vision.py
import base64
from openai import AsyncOpenAI
from playwright.async_api import Page

client = AsyncOpenAI()

async def screenshot_to_base64(page: Page) -> str:
    """Tira screenshot e converte para base64."""
    img_bytes = await page.screenshot(type="png", full_page=False)
    return base64.b64encode(img_bytes).decode("utf-8")

async def ask_vision(page: Page, question: str) -> str:
    """Pergunta ao GPT-4o Vision sobre o estado atual da página."""
    b64 = await screenshot_to_base64(page)

    response = await client.chat.completions.create(
        model="gpt-4o",
        messages=[{
            "role": "user",
            "content": [
                {
                    "type": "image_url",
                    "image_url": {
                        "url": f"data:image/png;base64,{b64}",
                        "detail": "high"
                    }
                },
                {"type": "text", "text": question}
            ]
        }],
        max_tokens=500
    )
    return response.choices[0].message.content

async def find_element_coords(page: Page, description: str) -> dict | None:
    """
    Usa Vision para encontrar coordenadas (x, y) de um elemento descrito
    em linguagem natural.
    """
    b64 = await screenshot_to_base64(page)

    # Obtém dimensões da página para normalizar
    vp = page.viewport_size
    w, h = vp["width"], vp["height"]

    prompt = f"""
    Na screenshot desta página web (resolução {w}x{h}px):

    Encontre: "{description}"

    Responda SOMENTE com JSON no formato:
    {{"found": true, "x": 350, "y": 120, "confidence": 0.95, "description": "botão verde 'Comprar'"}}

    Se não encontrar:
    {{"found": false, "reason": "elemento não visível na tela"}}

    Use coordenadas absolutas em pixels. Aponte para o centro do elemento.
    """

    response = await client.chat.completions.create(
        model="gpt-4o",
        messages=[{
            "role": "user",
            "content": [
                {"type": "image_url", "image_url": {"url": f"data:image/png;base64,{b64}"}},
                {"type": "text", "text": prompt}
            ]
        }],
        response_format={"type": "json_object"},
        max_tokens=200
    )

    import json
    result = json.loads(response.choices[0].message.content)
    return result if result.get("found") else None

class BrowserExecutor:
    """Executa as browser tools chamadas pelo agente."""

    def __init__(self, page: Page):
        self.page = page
        self.action_log: list[dict] = []

    async def execute(self, tool_name: str, args: dict) -> str:
        """Executa uma tool e retorna o resultado como string."""
        match tool_name:
            case "navigate":
                return await self._navigate(**args)
            case "click_element":
                return await self._click_element(**args)
            case "fill_input":
                return await self._fill_input(**args)
            case "read_page":
                return await self._read_page(**args)
            case "scroll_page":
                return await self._scroll_page(**args)
            case _:
                return f"Tool desconhecida: {tool_name}"

    async def _navigate(self, url: str) -> str:
        await self.page.goto(url, wait_until="networkidle")
        title = await self.page.title()
        self._log("navigate", {"url": url})
        return f"Navegou para {url}. Título da página: '{title}'"

    async def _click_element(self, description: str) -> str:
        coords = await find_element_coords(self.page, description)
        if not coords:
            return f"Elemento não encontrado: '{description}'"

        x, y = coords["x"], coords["y"]
        await self.page.mouse.click(x, y)
        await self.page.wait_for_load_state("networkidle")
        self._log("click", {"description": description, "coords": (x, y)})
        return f"Clicou em '{description}' em ({x}, {y}). Confiança: {coords.get('confidence', '?')}"

    async def _fill_input(self, description: str, text: str) -> str:
        coords = await find_element_coords(self.page, description)
        if not coords:
            return f"Campo não encontrado: '{description}'"

        x, y = coords["x"], coords["y"]
        await self.page.mouse.click(x, y)
        await self.page.keyboard.type(text, delay=50)
        self._log("fill", {"description": description, "text": text})
        return f"Preencheu '{description}' com '{text}'"

    async def _read_page(self, question: str) -> str:
        result = await ask_vision(self.page, question)
        return result

    async def _scroll_page(self, direction: str, amount: int = 500) -> str:
        delta = amount if direction == "down" else -amount
        await self.page.mouse.wheel(0, delta)
        await self.page.wait_for_timeout(500)
        return f"Scrollou {direction} {amount}px"

    def _log(self, action: str, details: dict):
        self.action_log.append({"action": action, **details})

ReAct Agent — O Loop Principal
Agente que recebe objetivo em linguagem natural e age autonomamente
agent.py
import json
import asyncio
from openai import AsyncOpenAI
from playwright.async_api import async_playwright
from browser_tools import BROWSER_TOOLS, BrowserExecutor

client = AsyncOpenAI()

SYSTEM_PROMPT = """Você é um agente de browser automation. Seu trabalho é completar
tarefas em páginas web usando as ferramentas disponíveis.

Abordagem:
1. Sempre leia a página atual antes de agir (use read_page)
2. Planeje seus próximos passos baseado no que você vê
3. Execute uma ação por vez
4. Verifique o resultado antes de continuar
5. Se uma ação falhar, tente uma abordagem alternativa

Nunca tente adivinhar elementos — use sempre read_page para ver o que está na tela.
Responda em português do Brasil."""


async def run_agent(objetivo: str, url_inicial: str, max_steps: int = 15):
    """
    Executa o agente de browser com objetivo em linguagem natural.

    Args:
        objetivo: O que o agente deve fazer (linguagem natural)
        url_inicial: URL onde começar
        max_steps: Máximo de ações antes de desistir

    Returns:
        dict com resultado e log de ações
    """
    async with async_playwright() as p:
        browser = await p.chromium.launch(headless=False)  # headless=True em produção
        page = await browser.new_page(viewport={"width": 1280, "height": 800})
        executor = BrowserExecutor(page)

        # Mensagens do conversation history
        messages = [
            {"role": "system", "content": SYSTEM_PROMPT},
            {
                "role": "user",
                "content": f"""
Objetivo: {objetivo}

Comece navegando para: {url_inicial}

Execute as ações necessárias para completar o objetivo.
Quando terminar, responda com 'OBJETIVO CONCLUÍDO: [resultado]'
"""
            }
        ]

        step = 0
        result = "Máximo de passos atingido"

        while step < max_steps:
            step += 1
            print(f"\n{'='*50}")
            print(f"PASSO {step}")

            # Chamar o LLM
            response = await client.chat.completions.create(
                model="gpt-4o",
                messages=messages,
                tools=BROWSER_TOOLS,
                tool_choice="auto",
                max_tokens=1000
            )

            msg = response.choices[0].message
            messages.append(msg)  # Adiciona ao histórico

            # Verificar se terminou
            if msg.content and "OBJETIVO CONCLUÍDO" in msg.content:
                result = msg.content
                print(f"✅ {result}")
                break

            # Verificar se há tool calls
            if not msg.tool_calls:
                print(f"💬 Agente: {msg.content}")
                result = msg.content
                break

            # Executar cada tool call
            tool_results = []
            for tool_call in msg.tool_calls:
                func_name = tool_call.function.name
                func_args = json.loads(tool_call.function.arguments)

                print(f"🔧 Chamando: {func_name}({func_args})")
                observation = await executor.execute(func_name, func_args)
                print(f"📊 Resultado: {observation[:200]}")

                tool_results.append({
                    "role": "tool",
                    "tool_call_id": tool_call.id,
                    "content": observation
                })

            messages.extend(tool_results)

        await browser.close()

        return {
            "objetivo": objetivo,
            "resultado": result,
            "passos": step,
            "acoes": executor.action_log
        }


# Exemplo de uso
async def main():
    resultado = await run_agent(
        objetivo="Pesquise 'Python asyncio' no Google e me diga o título do primeiro resultado",
        url_inicial="https://www.google.com"
    )
    print("\n" + "="*60)
    print("RELATÓRIO FINAL")
    print("="*60)
    print(f"Objetivo: {resultado['objetivo']}")
    print(f"Passos executados: {resultado['passos']}")
    print(f"Resultado: {resultado['resultado']}")
    print(f"Ações: {json.dumps(resultado['acoes'], indent=2, ensure_ascii=False)}")

if __name__ == "__main__":
    asyncio.run(main())

Research Agent — Coletando Dados Estruturados

Um caso de uso poderoso: agentes de pesquisa automática que navegam em múltiplas páginas, coletam dados e retornam JSON estruturado. Combinamos Vision LLM com extração via Pydantic para garantir a qualidade dos dados.

research_agent.py — Pesquisa de preços multi-site
import asyncio
import json
from pydantic import BaseModel, Field
from typing import Optional
from openai import AsyncOpenAI
from playwright.async_api import async_playwright, Page
from vision import ask_vision, screenshot_to_base64

client = AsyncOpenAI()


class ProductData(BaseModel):
    """Schema para produto coletado via Vision."""
    nome: str
    preco: Optional[float] = None
    preco_original: Optional[float] = None
    disponivel: bool = True
    avaliacao: Optional[float] = None
    num_avaliacoes: Optional[int] = None
    descricao: Optional[str] = None
    url: str


async def extract_product_vision(page: Page) -> ProductData:
    """
    Extrai dados de produto usando Vision LLM.
    Funciona em qualquer loja — sem seletores específicos.
    """
    b64 = await screenshot_to_base64(page)
    url = page.url

    prompt = """
    Analise esta página de produto e extraia as informações no formato JSON:

    {
      "nome": "nome completo do produto",
      "preco": 99.90,
      "preco_original": 149.90,
      "disponivel": true,
      "avaliacao": 4.5,
      "num_avaliacoes": 1234,
      "descricao": "breve descrição"
    }

    Use null para campos não encontrados. Preco deve ser número float.
    Responda SOMENTE com o JSON, sem markdown.
    """

    response = await client.chat.completions.create(
        model="gpt-4o",
        messages=[{
            "role": "user",
            "content": [
                {"type": "image_url", "image_url": {"url": f"data:image/png;base64,{b64}"}},
                {"type": "text", "text": prompt}
            ]
        }],
        response_format={"type": "json_object"},
        max_tokens=400
    )

    data = json.loads(response.choices[0].message.content)
    return ProductData(url=url, **data)


async def research_product_prices(produto: str, lojas: list[str]) -> list[ProductData]:
    """
    Pesquisa preços de um produto em múltiplas lojas usando Vision.

    Args:
        produto: Nome do produto para pesquisar
        lojas: Lista de URLs de lojas para pesquisar

    Returns:
        Lista de ProductData com dados coletados de cada loja
    """
    resultados = []

    async with async_playwright() as p:
        browser = await p.chromium.launch()
        context = await browser.new_context(
            user_agent="Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36",
            viewport={"width": 1280, "height": 800}
        )

        for loja_url in lojas:
            print(f"🔍 Pesquisando em {loja_url}...")
            page = await context.new_page()

            try:
                # Navegar para a loja
                await page.goto(loja_url, wait_until="networkidle", timeout=30000)

                # Usar Vision para encontrar e clicar na busca
                coords = await find_search_box(page)
                if coords:
                    await page.mouse.click(coords["x"], coords["y"])
                    await page.keyboard.type(produto, delay=80)
                    await page.keyboard.press("Enter")
                    await page.wait_for_load_state("networkidle")

                # Encontrar o primeiro resultado
                first_result = await find_first_product(page, produto)
                if first_result:
                    await page.mouse.click(first_result["x"], first_result["y"])
                    await page.wait_for_load_state("networkidle")

                    # Extrair dados do produto
                    produto_data = await extract_product_vision(page)
                    resultados.append(produto_data)
                    print(f"  ✅ {produto_data.nome}: R$ {produto_data.preco}")
                else:
                    print(f"  ⚠️  Produto não encontrado em {loja_url}")

            except Exception as e:
                print(f"  ❌ Erro: {e}")
            finally:
                await page.close()

        await browser.close()

    # Ordenar por preço
    return sorted(resultados, key=lambda p: p.preco or float('inf'))


async def find_search_box(page: Page) -> dict | None:
    """Localiza campo de busca via Vision."""
    from vision import find_element_coords
    return await find_element_coords(
        page,
        "campo de busca ou barra de pesquisa no topo da página"
    )


async def find_first_product(page: Page, produto: str) -> dict | None:
    """Localiza o primeiro produto relevante nos resultados."""
    from vision import find_element_coords
    return await find_element_coords(
        page,
        f"primeiro resultado de produto relacionado a '{produto}' na lista de resultados"
    )


# Executar pesquisa
async def main():
    resultados = await research_product_prices(
        produto="iPhone 15 128GB",
        lojas=[
            "https://www.magazineluiza.com.br",
            "https://www.americanas.com.br",
            "https://www.casasbahia.com.br",
        ]
    )

    print("\n📊 COMPARATIVO DE PREÇOS")
    print("="*50)
    for i, r in enumerate(resultados, 1):
        economia = ""
        if r.preco_original and r.preco and r.preco_original > r.preco:
            economia = f" (-{((r.preco_original - r.preco)/r.preco_original*100):.0f}%)"
        print(f"{i}. R$ {r.preco:.2f}{economia} — {r.url}")

if __name__ == "__main__":
    asyncio.run(main())
⚠️ Rate Limits e Custos

pipeline.py — Monitor de preços completo
import asyncio
import sqlite3
from datetime import datetime
from dataclasses import dataclass
import httpx
from research_agent import extract_product_vision, ProductData
from playwright.async_api import async_playwright

N8N_WEBHOOK_URL = "https://n8n.seu-dominio.com/webhook/price-alert"
ALERT_THRESHOLD = 0.10  # 10% de queda


@dataclass
class PriceRecord:
    produto_url: str
    preco: float
    timestamp: str


class PriceMonitorDB:
    def __init__(self, db_path: str = "prices.db"):
        self.conn = sqlite3.connect(db_path)
        self._create_tables()

    def _create_tables(self):
        self.conn.execute("""
            CREATE TABLE IF NOT EXISTS price_history (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                produto_url TEXT NOT NULL,
                nome TEXT,
                preco REAL NOT NULL,
                timestamp TEXT NOT NULL
            )
        """)
        self.conn.commit()

    def get_last_price(self, url: str) -> float | None:
        cur = self.conn.execute(
            "SELECT preco FROM price_history WHERE produto_url=? ORDER BY timestamp DESC LIMIT 1",
            (url,)
        )
        row = cur.fetchone()
        return row[0] if row else None

    def save_price(self, produto: ProductData):
        self.conn.execute(
            "INSERT INTO price_history (produto_url, nome, preco, timestamp) VALUES (?,?,?,?)",
            (produto.url, produto.nome, produto.preco, datetime.now().isoformat())
        )
        self.conn.commit()


async def send_price_alert(produto: ProductData, preco_anterior: float):
    """Envia alerta via n8n webhook."""
    queda_pct = (preco_anterior - produto.preco) / preco_anterior * 100

    payload = {
        "produto": produto.nome,
        "url": produto.url,
        "preco_atual": produto.preco,
        "preco_anterior": preco_anterior,
        "queda_percentual": round(queda_pct, 1),
        "timestamp": datetime.now().isoformat()
    }

    async with httpx.AsyncClient() as client:
        resp = await client.post(N8N_WEBHOOK_URL, json=payload)
        print(f"🚨 Alerta enviado! Status: {resp.status_code}")


async def monitor_url(url: str, db: PriceMonitorDB):
    """Monitora uma URL específica."""
    async with async_playwright() as p:
        browser = await p.chromium.launch()
        page = await browser.new_page()
        await page.goto(url, wait_until="networkidle")

        produto = await extract_product_vision(page)
        await browser.close()

    if not produto.preco:
        print(f"⚠️  Sem preço para {url}")
        return

    preco_anterior = db.get_last_price(url)
    db.save_price(produto)

    print(f"📦 {produto.nome}: R$ {produto.preco:.2f}")

    if preco_anterior:
        queda = (preco_anterior - produto.preco) / preco_anterior
        if queda >= ALERT_THRESHOLD:
            await send_price_alert(produto, preco_anterior)
        else:
            print(f"  (anterior: R$ {preco_anterior:.2f}, variação: {queda*100:+.1f}%)")


async def run_monitor(urls: list[str]):
    """Executa monitoramento em todas as URLs."""
    db = PriceMonitorDB()

    # Monitorar em paralelo com semáforo (máx 3 simultâneos)
    sem = asyncio.Semaphore(3)

    async def monitor_with_sem(url):
        async with sem:
            await monitor_url(url, db)

    await asyncio.gather(*[monitor_with_sem(url) for url in urls])


if __name__ == "__main__":
    urls_para_monitorar = [
        "https://www.loja.com/produto/iphone-15",
        "https://www.loja.com/produto/macbook-air",
    ]
    asyncio.run(run_monitor(urls_para_monitorar))

Armadilhas Comuns

Pitfalls a Evitar

⚠️ Screenshots em alta resolução consomem muito token

Uma screenshot fullpage de 1920x1080 usa ~1500 tokens de imagem. Use full_page=False e detail: "low" sempre que possível. Considere recortar apenas a área relevante com page.screenshot(clip={"x": 0, "y": 0, "width": 800, "height": 600}).

⚠️ Coordenadas incorretas em telas com deviceScaleFactor

Em Retina/HiDPI, a screenshot tem 2x a resolução do viewport. Se o GPT-4o retornar coordenadas baseadas na imagem (2x), você precisa dividir por deviceScaleFactor antes de passar para page.mouse.click(). Solução: sempre usar new_context(device_scale_factor=1).

⚠️ Agente em loop infinito

Sem um max_steps rígido, um agente pode executar 50+ chamadas ao LLM se ficar confuso. Sempre implemente limite de passos e timeout total. Monitore custo com usage.total_tokens em cada resposta.

⚠️ Pop-ups e cookies bloqueando a visão

Banners de cookies e pop-ups aparecem em screenshots e confundem o Vision LLM. Adicione um passo inicial: pedir ao agente para "fechar qualquer pop-up ou banner de cookies visível" antes de executar o objetivo principal.

browser-use — Abstração de Alto Nível

Para quem não quer implementar todo o loop do zero, o browser-use é uma biblioteca open-source que encapsula exatamente a arquitetura que vimos — Playwright + Vision LLM + ReAct — com uma API limpa.

exemplo com browser-use
import asyncio
from browser_use import Agent
from langchain_openai import ChatOpenAI


async def main():
    # Setup simples — browser-use cuida de todo o loop
    agent = Agent(
        task="Vá para github.com/trending e me liste os 5 repositórios mais populares desta semana com suas estrelas",
        llm=ChatOpenAI(model="gpt-4o"),
    )

    result = await agent.run()
    print(result)

    # Uso com histórico persistente
    agent2 = Agent(
        task="Pesquise 'FastAPI tutorial' no YouTube e me dê o título e duração do primeiro vídeo",
        llm=ChatOpenAI(model="gpt-4o"),
        max_actions_per_step=5,  # Limite por passo
    )
    result2 = await agent2.run(max_steps=10)
    print(result2)

asyncio.run(main())

🏁 Checkpoint — Desafio Final

Construa um Vision Agent de Pesquisa que, dado um produto em linguagem natural, navega até o Google Shopping, coleta dados dos 3 primeiros resultados (nome, preço, loja) e retorna um JSON comparativo. O agente deve funcionar sem seletores CSS hardcoded.

  • Use find_element_coords() para localizar a barra de busca
  • Use extract_product_vision() adaptado para resultados de busca
  • Valide o output com Pydantic (list[ProductData])
  • Salve o resultado em resultado_{timestamp}.json
  • Adicione retry com backoff exponencial para falhas de Vision
💬 Dica: Google Shopping pode exigir scroll para mostrar mais resultados. Use scroll_page("down", 500) e um segundo read_page() para capturar todos os resultados. Considere usar detail: "high" para leitura de preços (pequenos e precisos) e detail: "low" para navegação geral.
