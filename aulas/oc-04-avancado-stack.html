Anti-bot: Como Sites Detectam e Como Contornar

Sites modernos usam múltiplas camadas para detectar automação. Entender o que estão verificando é o primeiro passo para contornar eticamente.

🕵️ User-Agent e Fingerprinting
Headless browsers têm fingerprints distintos: navigator.webdriver=true, ausência de plugins, canvas fingerprint diferente.
✅ Solução: playwright-stealth + rotação de user-agents reais + injetar navigator.webdriver=undefined
⏱️ Velocidade de Interação
Bots clicam instantaneamente; humanos têm delays naturais. Movimento do mouse em linha reta é impossível humanamente.
✅ Solução: typing delays, mouse movement curvado, waits aleatórios entre ações (1.5s-4s)
🍪 Cookies e Sessions
Bots sem histórico de cookies parecem suspeitos. Sem cookies de sessão de visitas anteriores.
✅ Solução: warm-up sessions (visitar homepage antes da página alvo), reutilizar auth_state.json entre runs
🔐 CAPTCHA e hCaptcha
reCAPTCHA v3 é score-based (sem desafio), hCaptcha requer cliques, Cloudflare Turnstile é challenge automático.
✅ Solução: 2captcha API / CapSolver para resolução automática, ou proxies residenciais que já têm score alto
🌐 Rate Limiting por IP
Muitas requisições do mesmo IP em pouco tempo → ban temporário ou permanente.
✅ Solução: pool de proxies residenciais, delays adaptativos, respeitar robots.txt e limitar a dados públicos

# JavaScript para mascarar fingerprint de automação
STEALTH_SCRIPTS = """
    // Remove webdriver
    Object.defineProperty(navigator, 'webdriver', {
        get: () => undefined,
    });

    // Adicionar plugins simulados
    Object.defineProperty(navigator, 'plugins', {
        get: () => [
            {name: 'Chrome PDF Plugin', filename: 'internal-pdf-viewer'},
            {name: 'Chrome PDF Viewer', filename: 'mhjfbmdgcfjbbpaeojofohoefgiehjai'},
            {name: 'Native Client', filename: 'internal-nacl-plugin'},
        ],
    });

        loadTimes: function(){ return {}; }
    };

    // Simular permissões normais
    const originalQuery = window.navigator.permissions.query;
    window.navigator.permissions.query = (parameters) => (

async def movimento_humano(page: Page, dest_x: float, dest_y: float):
    """
    Move o mouse em curva Bezier simulando movimento humano.
    Bots movem em linha reta; humanos fazem curvas naturais.
    """
    pos = await page.evaluate("() => ({x: 0, y: 0})")
    start_x, start_y = pos["x"], pos["y"]

    # Ponto de controle aleatório para curva Bezier
    ctrl_x = (start_x + dest_x) / 2 + random.uniform(-100, 100)
    ctrl_y = (start_y + dest_y) / 2 + random.uniform(-80, 80)

    steps = random.randint(20, 40)
    for i in range(steps + 1):
        t = i / steps
        # Fórmula Bezier quadrática
        x = (1-t)**2 * start_x + 2*(1-t)*t * ctrl_x + t**2 * dest_x
        y = (1-t)**2 * start_y + 2*(1-t)*t * ctrl_y + t**2 * dest_y

        await page.mouse.move(x, y)
        await asyncio.sleep(random.uniform(0.005, 0.015))

    await asyncio.sleep(random.uniform(0.08, 0.2))


async def digitar_humano(page: Page, selector: str, texto: str):
    """
    Digita texto com velocidade variável simulando digitação humana.
    """
    await page.click(selector)
    await asyncio.sleep(random.uniform(0.3, 0.7))

    for char in texto:
        await page.keyboard.type(char)
        # Velocidade varia: caracteres comuns mais rápidos, números/especiais mais lentos
        delay = random.uniform(0.06, 0.15)
        if char in "0123456789@.!?":
            delay *= 1.5
        await asyncio.sleep(delay)

    await asyncio.sleep(random.uniform(0.2, 0.5))


@asynccontextmanager
async def stealth_context(proxy_url: str | None = None):
    """
    Cria contexto de browser com todas as técnicas anti-detecção aplicadas.
    """
    async with async_playwright() as p:
        launch_args = {
            "headless": True,
            "args": [
                "--disable-blink-features=AutomationControlled",
                "--no-sandbox",
                "--disable-web-security",
                "--disable-features=IsolateOrigins",
                f"--window-size={random.choice([1366, 1440, 1920])},{random.choice([768, 900, 1080])}",
            ]
        }

        if proxy_url:
            launch_args["proxy"] = {"server": proxy_url}

        browser = await p.chromium.launch(**launch_args)

        context_args = {
            "user_agent": random.choice(USER_AGENTS),
            "viewport": {"width": 1366, "height": 768},
            "locale": "pt-BR",
            "timezone_id": "America/Sao_Paulo",
            "permissions": ["geolocation"],
            "extra_http_headers": {
                "Accept-Language": "pt-BR,pt;q=0.9,en-US;q=0.8,en;q=0.7",
                "Accept-Encoding": "gzip, deflate, br",
                "DNT": "1",
            }
        }

        context: BrowserContext = await browser.new_context(**context_args)

        # Injetar script de stealth em todas as páginas
        await context.add_init_script(STEALTH_SCRIPTS)

        try:
            yield context
        finally:
            await browser.close()


async def warm_up_session(context: BrowserContext, dominio: str):
    """
    Simula visita inicial ao site para construir cookies legítimos
    antes de fazer scraping.
    """
    page = await context.new_page()

    # Visitar homepage
    await page.goto(f"https://{dominio}", wait_until="networkidle")
    await asyncio.sleep(random.uniform(2, 4))

    # Scrollar para parecer que está lendo
    for _ in range(random.randint(2, 4)):
        await page.mouse.wheel(0, random.randint(300, 700))
        await asyncio.sleep(random.uniform(1, 2.5))

    await page.close()


# Uso prático
async def scrape_com_stealth(url: str) -> str:
    """Exemplo de scraping com todas as proteções aplicadas."""
    async with stealth_context() as context:
        from urllib.parse import urlparse
        dominio = urlparse(url).netloc

        # Warm-up antes de scraping
        await warm_up_session(context, dominio)

        page = await context.new_page()
        await page.goto(url, wait_until="networkidle")

        # Delay humano antes de interagir
        await asyncio.sleep(random.uniform(1.5, 3))

        content = await page.content()
        await page.close()
        return content

workers.py — Sistema de workers com Redis Queue
import asyncio
import json
from datetime import datetime
from redis import Redis
from rq import Queue, Worker
from rq.job import Job
import psycopg2
from dataclasses import dataclass, asdict
from typing import Optional


# Configuração Redis
redis_conn = Redis(host="localhost", port=6379, db=0)
scraping_queue = Queue("scraping", connection=redis_conn)
priority_queue = Queue("priority", connection=redis_conn)


@dataclass
class ScrapeJob:
    url: str
    produto_id: str
    prioridade: int = 0
    max_retries: int = 3
    notify_on_change: bool = True


class ScrapingService:
    """Serviço principal de scraping com filas."""

    def __init__(self):
        self.redis = redis_conn
        self.db = self._conectar_db()

    def _conectar_db(self):
        return psycopg2.connect(
            host="localhost", database="scraping_db",
            user="postgres", password="senha"
        )

    def agendar_scraping(self, job: ScrapeJob) -> str:
        """Adiciona URL na fila de scraping."""
        queue = priority_queue if job.prioridade > 5 else scraping_queue

        rq_job = queue.enqueue(
            "workers.executar_scraping",  # função a executar
            kwargs=asdict(job),
            job_timeout=120,
            retry=job.max_retries,
            result_ttl=3600
        )

        print(f"📥 Job {rq_job.id} agendado para {job.url}")
        return rq_job.id

    def agendar_lote(self, urls: list[str], produto_ids: list[str]):
        """Agenda múltiplas URLs de uma vez."""
        job_ids = []
        for url, pid in zip(urls, produto_ids):
            job = ScrapeJob(url=url, produto_id=pid)
            job_id = self.agendar_scraping(job)
            job_ids.append(job_id)
        return job_ids

    def status_fila(self) -> dict:
        """Retorna status atual das filas."""
        return {
            "scraping": {
                "pendentes": len(scraping_queue),
                "falhados": scraping_queue.failed_job_registry.count,
                "concluidos": scraping_queue.finished_job_registry.count,
            },
            "priority": {
                "pendentes": len(priority_queue),
            }
        }


# Função executada pelos workers (deve ser importável pelo RQ)
def executar_scraping(url: str, produto_id: str, notify_on_change: bool = True, **kwargs):
    """
    Função executada por cada worker.
    Note: não pode ser async — RQ usa threads síncrono.
    Use asyncio.run() para código async.
    """
    from stealth_browser import scrape_com_stealth
    from research_agent import extract_product_vision
    from playwright.sync_api import sync_playwright

    print(f"🔧 Worker processando: {url}")

    # Executar scraping
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=True)
        page = browser.new_page()
        page.goto(url, wait_until="networkidle")

        # Extrair dados (versão síncrona simplificada)
        titulo = page.title()
        preco_text = page.locator("[data-price], .price, .preco").first.inner_text()

        import re
        preco_match = re.search(r'[\d.,]+', preco_text.replace(".", "").replace(",", "."))
        preco = float(preco_match.group()) if preco_match else None

        browser.close()

    resultado = {
        "url": url,
        "produto_id": produto_id,
        "titulo": titulo,
        "preco": preco,
        "timestamp": datetime.now().isoformat()
    }

    # Salvar no banco e verificar mudança
    salvar_resultado(resultado)
    if notify_on_change and preco:
        verificar_e_notificar(produto_id, preco)

    return resultado


def salvar_resultado(dados: dict):
    """Salva resultado no PostgreSQL."""
    conn = psycopg2.connect(
        host="localhost", database="scraping_db",
        user="postgres", password="senha"
    )
    with conn.cursor() as cur:
        cur.execute("""
            INSERT INTO price_history (produto_id, url, preco, titulo, coletado_em)
            VALUES (%(produto_id)s, %(url)s, %(preco)s, %(titulo)s, %(timestamp)s)
        """, dados)
    conn.commit()
    conn.close()


def verificar_e_notificar(produto_id: str, preco_atual: float):
    """Verifica se preço mudou e dispara webhook n8n se necessário."""
    import httpx

    conn = psycopg2.connect(
        host="localhost", database="scraping_db",
        user="postgres", password="senha"
    )
    with conn.cursor() as cur:
        cur.execute("""
            SELECT preco FROM price_history
            WHERE produto_id = %s
            ORDER BY coletado_em DESC
            LIMIT 2
        """, (produto_id,))
        rows = cur.fetchall()
    conn.close()

    if len(rows) >= 2:
        preco_anterior = rows[1][0]
        variacao = (preco_atual - preco_anterior) / preco_anterior

        if abs(variacao) >= 0.05:  # Variação >= 5%
            httpx.post(
                "http://n8n:5678/webhook/price-alert",
                json={
                    "produto_id": produto_id,
                    "preco_atual": preco_atual,
                    "preco_anterior": float(preco_anterior),
                    "variacao_pct": round(variacao * 100, 1)
                }
            )

Pitfalls em Produção

⚠️ Workers sem memória — acúmulo de processos Chromium

Playwright em workers pode vazar memória se o browser não for fechado corretamente em caso de erro. Use sempre try/finally ou context managers para garantir browser.close(). Adicione limite de memória no Docker: mem_limit: 512m.

⚠️ Ban de IP em produção

Se o IP do seu servidor for banido, todos os workers param de funcionar. Solução: use um VPS com IP diferente do development, implemente backoff exponencial quando detectar bloqueio (resposta 403/429), e considere proxies rotativos para alta frequência.

⚠️ Jobs presos na fila

Jobs que falham silenciosamente ficam em "started" infinitamente. Configure job_timeout em todos os jobs e monitore started_job_registry do RQ. Use RQ Dashboard para inspecionar e reencaminhar jobs travados.

🏁 Projeto Final do Curso

Sistema de Inteligência de Mercado — Integre tudo o que aprendeu nas 50 aulas para construir um sistema completo de automação inteligente.

  • Coleta: Playwright com stealth monitorando 20+ produtos em 3 lojas
  • Extração: Pydantic v2 com validação + fallback via Vision LLM (GPT-4o)
  • Escala: Redis Queue com 3 workers Docker rodando em paralelo
  • Banco: PostgreSQL com histórico de preços e análise de tendências
  • Alertas: n8n webhook → Telegram/Slack quando preço cai >10%
  • IA: Agente CrewAI que gera relatório semanal de inteligência competitiva
  • CI/CD: GitHub Actions com testes (pytest) e deploy automático via SSH
  • Observabilidade: Grafana com métricas de jobs, latência e erros
💬 Dica: Comece pelo core (Playwright + PostgreSQL + uma loja), garanta que funciona, depois adicione camada por camada: workers, n8n, IA, monitoramento. Código incremental é mais robusto que big-bang. Use o template Docker Compose desta aula como base — já tem tudo configurado.
