Por que isso importa?

Playwright resolve o "como" da automação. OpenClaw resolve o "o quê" — ele é uma camada de alto nível construída sobre Playwright que fornece tarefas prontas, pipelines reutilizáveis e integração com IA para as situações mais comuns de automação de browser.

Pense assim: Playwright é o motor; OpenClaw é o carro completo. Com o motor você pode fazer qualquer coisa, mas o carro te dá volante, câmbio, espelhos, GPS — tudo pronto. Um engenheiro sênior usa OpenClaw para entregar pipelines de automação em horas, não em dias.

💡 O que é OpenClaw?
OpenClaw é um framework de automação de browser construído sobre Playwright, focado em três pilares: tarefas estruturadas (actions reutilizáveis), scraping inteligente (extração tipada com validação) e integração com IA (visão computacional + LLMs para navegar em páginas complexas). Esta aula cobre o setup e as tarefas core; a próxima (Aula 48) aborda a camada de IA.
📋

Tasks Declarativas

Descreve "o que" fazer, não "como" fazer. OpenClaw cuida dos waits, retries e erros automaticamente.

🔐

Auth Gerenciada

Login, sessão, cookies e tokens gerenciados automaticamente — autentique uma vez, reutilize em todas as tasks.

🔄

Pipelines Compostos

Encadeie tasks em pipelines: login → navegar → extrair → validar → exportar. Tudo rastreável.

🛡

Anti-Detection

Fingerprinting realista, delays humanizados e rotação de user-agents embutidos por padrão.

Browser Factory — Instância Configurada e Reutilizável

Em vez de configurar o browser em cada script, centralize toda a configuração (stealth, viewport, user-agent) num único factory reutilizável.

python — browser_factory.py
import asyncio
from contextlib import asynccontextmanager
from playwright.async_api import async_playwright, Browser, BrowserContext, Page
from config import settings

@asynccontextmanager
async def novo_contexto(browser: Browser):
    """Cria um contexto isolado com configurações anti-detection."""
    contexto = await browser.new_context(
        viewport={"width": settings.viewport_width, "height": settings.viewport_height},
        user_agent=settings.user_agent,
        locale=settings.locale,
        timezone_id=settings.timezone,
        # Simula permissões reais do browser
        permissions=["geolocation"],
        # Headers extras para parecer mais real
        extra_http_headers={
            "Accept-Language": "pt-BR,pt;q=0.9,en;q=0.8",
            "Accept": "text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8",
        }
    )
    # Ocultar flag webdriver (detecção básica de bot)
    await contexto.add_init_script("""
        Object.defineProperty(navigator, 'webdriver', { get: () => undefined });
        Object.defineProperty(navigator, 'plugins', { get: () => [1, 2, 3] });
        window.chrome = { runtime: {} };
    """)
    try:
        yield contexto
    finally:
        await contexto.close()

@asynccontextmanager
async def nova_sessao():
    """Context manager completo: Playwright → Browser → Context → Page."""
    async with async_playwright() as p:
        browser = await p.chromium.launch(
            headless=settings.headless,
            slow_mo=settings.slow_mo,
            args=[
                "--no-sandbox",
                "--disable-blink-features=AutomationControlled",
                "--disable-web-security",
            ]
        )
        async with novo_contexto(browser) as contexto:
            page = await contexto.new_page()
            page.set_default_timeout(settings.timeout)
            try:
                yield page, contexto
            finally:
                await browser.close()

Autenticação Gerenciada — Login uma vez, reutilize sempre

O padrão mais importante em projetos de automação: fazer login uma única vez, salvar o estado da sessão (cookies + localStorage) em disco, e reutilizar em todas as execuções seguintes — evitando logins repetidos e reduzindo chances de detecção.

// fluxo de autenticação com cache
flowchart TD A([Iniciar sessão]) --> B{auth_state.json\nexiste?} B -->|Sim| C[Carregar estado salvo] C --> D{Sessão ainda\nválida?} D -->|Sim| G([✅ Pronto para usar]) D -->|Não — expirou| E B -->|Não| E[Executar login\ncompleto] E --> F[Salvar estado\nem auth_state.json] F --> G
python — tasks/auth.py
import json
from pathlib import Path
from playwright.async_api import BrowserContext, Page
from config import settings

AUTH_STATE_FILE = Path("./auth_state.json")

async def fazer_login(page: Page, url_login: str) -> bool:
    """Executa o fluxo de login no site."""
    try:
        await page.goto(url_login)
        await page.wait_for_selector("#email, input[type='email']")

        await page.fill("#email", settings.site_usuario)
        await page.fill("#password", settings.site_senha)

        # Delay humanizado antes de clicar
        await page.wait_for_timeout(800)
        await page.click("button[type='submit']")

        # Aguarda confirmação de login (URL muda ou elemento aparece)
        await page.wait_for_url("**/dashboard", timeout=15_000)
        print("✅ Login realizado com sucesso")
        return True

    except Exception as e:
        print(f"❌ Falha no login: {e}")
        return False

async def salvar_estado_auth(contexto: BrowserContext) -> None:
    """Salva cookies e localStorage em disco."""
    estado = await contexto.storage_state()
    AUTH_STATE_FILE.write_text(json.dumps(estado, ensure_ascii=False))
    print(f"💾 Estado de autenticação salvo em {AUTH_STATE_FILE}")

async def carregar_estado_auth(contexto: BrowserContext) -> bool:
    """Carrega estado de autenticação salvo anteriormente."""
    if not AUTH_STATE_FILE.exists():
        return False
    estado = json.loads(AUTH_STATE_FILE.read_text())
    await contexto.add_cookies(estado.get("cookies", []))
    print("🔑 Estado de autenticação carregado")
    return True

async def verificar_autenticado(page: Page, selector_logado: str = ".user-menu") -> bool:
    """Verifica se a sessão ainda é válida."""
    try:
        await page.goto("https://site.com/dashboard")
        await page.wait_for_selector(selector_logado, timeout=5_000)
        return True
    except Exception:
        return False

async def garantir_autenticado(page: Page, contexto: BrowserContext, url_login: str) -> bool:
    """Fluxo completo: carrega cache → verifica → faz login se necessário."""
    # Tentar carregar sessão salva
    if await carregar_estado_auth(contexto):
        if await verificar_autenticado(page):
            print("✅ Sessão reutilizada do cache")
            return True
        print("⚠ Sessão expirada, fazendo novo login...")

    # Fazer login e salvar
    sucesso = await fazer_login(page, url_login)
    if sucesso:
        await salvar_estado_auth(contexto)
    return sucesso

Scraping Estruturado com Pydantic

Cada item extraído é validado contra um schema Pydantic. Dados inválidos são descartados ou corrigidos automaticamente — nunca chegam corrompidos ao banco de dados.

python — models/produto.py
from pydantic import BaseModel, field_validator, HttpUrl
from typing import Optional
from datetime import datetime

class Produto(BaseModel):
    id: str
    nome: str
    preco: float
    preco_original: Optional[float] = None
    desconto_pct: Optional[float] = None
    avaliacao: Optional[float] = None
    num_avaliacoes: int = 0
    em_estoque: bool = True
    url: str
    coletado_em: datetime = datetime.now()

    @field_validator("preco", "preco_original", mode="before")
    @classmethod
    def limpar_preco(cls, v):
        """Converte 'R$ 1.299,90' → 1299.90"""
        if isinstance(v, str):
            return float(
                v.replace("R$", "").replace(".", "").replace(",", ".").strip()
            )
        return v

    @field_validator("avaliacao", mode="before")
    @classmethod
    def limpar_avaliacao(cls, v):
        if isinstance(v, str):
            return float(v.replace(",", ".").strip() or 0)
        return v

    def model_post_init(self, __context):
        if self.preco_original and self.preco_original > self.preco:
            self.desconto_pct = round(
                (1 - self.preco / self.preco_original) * 100, 1
            )
python — tasks/extraction.py
import asyncio
from playwright.async_api import Page
from models.produto import Produto
from config import settings

async def extrair_produto(item) -> Produto | None:
    """Extrai dados de um card de produto com tratamento robusto de erros."""
    try:
        id_elem = await item.get_attribute("data-product-id") or ""
        nome   = (await item.locator(".product-title").text_content() or "").strip()
        preco  = (await item.locator(".price-current").text_content() or "0").strip()

        preco_orig_el = item.locator(".price-original")
        preco_orig = None
        if await preco_orig_el.count() > 0:
            preco_orig = (await preco_orig_el.text_content() or "").strip()

        avaliacao_el = item.locator("[data-rating]")
        avaliacao = None
        if await avaliacao_el.count() > 0:
            avaliacao = await avaliacao_el.get_attribute("data-rating")

        num_av_el = item.locator(".review-count")
        num_avaliacoes = "0"
        if await num_av_el.count() > 0:
            num_avaliacoes = await num_av_el.text_content() or "0"
        num_avaliacoes = int(''.join(filter(str.isdigit, num_avaliacoes)) or 0)

        url_el = item.locator("a[href]").first
        url = await url_el.get_attribute("href") or ""
        if url.startswith("/"):
            url = f"https://loja.exemplo.com{url}"

        estoque_el = item.locator(".out-of-stock")
        em_estoque = await estoque_el.count() == 0

        return Produto(
            id=id_elem,
            nome=nome,
            preco=preco,
            preco_original=preco_orig,
            avaliacao=avaliacao,
            num_avaliacoes=num_avaliacoes,
            em_estoque=em_estoque,
            url=url
        )
    except Exception as e:
        print(f"  ⚠ Produto inválido ignorado: {e}")
        return None

async def extrair_pagina_produtos(page: Page) -> list[Produto]:
    """Extrai todos os produtos válidos da página atual."""
    await page.wait_for_selector(".product-card", state="visible")
    items = await page.locator(".product-card").all()

    # Extrair em paralelo (limitado para não sobrecarregar)
    semaphore = asyncio.Semaphore(5)

    async def extrair_com_limite(item):
        async with semaphore:
            return await extrair_produto(item)

    resultados = await asyncio.gather(*[extrair_com_limite(i) for i in items])
    validos = [r for r in resultados if r is not None]
    print(f"  ✓ {len(validos)}/{len(items)} produtos extraídos com sucesso")
    return validos

python — pipelines/scrape_produtos.py
import asyncio, json, csv
from dataclasses import asdict
from pathlib import Path
from browser_factory import nova_sessao
from tasks.auth import garantir_autenticado
from tasks.extraction import extrair_pagina_produtos
from config import settings

async def pipeline_scrape_produtos(
    url_categoria: str,
    max_paginas: int = 20
) -> list[dict]:
    """Pipeline completo de scraping autenticado com paginação."""

    todos_produtos = []

    async with nova_sessao() as (page, contexto):
        print("🔐 Verificando autenticação...")
        autenticado = await garantir_autenticado(
            page, contexto,
            url_login="https://loja.exemplo.com/login"
        )
        if not autenticado:
            raise RuntimeError("Falha na autenticação — abortando pipeline")

        print(f"🌐 Navegando para: {url_categoria}")
        await page.goto(url_categoria)

        for num_pagina in range(1, max_paginas + 1):
            print(f"\n📄 Processando página {num_pagina}...")
            produtos = await extrair_pagina_produtos(page)
            todos_produtos.extend([p.model_dump() for p in produtos])

            # Verificar paginação
            proximo = page.locator("a.pagination-next:not(.disabled), button.next-page")
            if await proximo.count() == 0 or not await proximo.is_enabled():
                print(f"✅ Última página alcançada ({num_pagina} páginas processadas)")
                break

            await proximo.click()
            await page.wait_for_load_state("networkidle")

            # Delay humanizado entre páginas
            await asyncio.sleep(1.5 + asyncio.get_event_loop().time() % 1)

    return todos_produtos

async def salvar_resultados(produtos: list[dict], nome_arquivo: str = "produtos"):
    """Salva em JSON e CSV."""
    saida = settings.output_dir

    # JSON
    json_path = saida / f"{nome_arquivo}.json"
    json_path.write_text(
        json.dumps(produtos, ensure_ascii=False, indent=2, default=str)
    )

    # CSV
    if produtos:
        csv_path = saida / f"{nome_arquivo}.csv"
        with csv_path.open("w", newline="", encoding="utf-8") as f:
            writer = csv.DictWriter(f, fieldnames=produtos[0].keys())
            writer.writeheader()
            writer.writerows(produtos)

    print(f"\n💾 {len(produtos)} produtos salvos:")
    print(f"   JSON → {json_path}")
    print(f"   CSV  → {csv_path}")

async def main():
    produtos = await pipeline_scrape_produtos(
        url_categoria="https://loja.exemplo.com/eletronicos",
        max_paginas=5
    )
    await salvar_resultados(produtos, "eletronicos")

    # Estatísticas rápidas
    if produtos:
        precos = [p["preco"] for p in produtos if p.get("preco")]
        print(f"\n📊 Resumo:")
        print(f"   Total de produtos: {len(produtos)}")
        print(f"   Preço mínimo: R$ {min(precos):.2f}")
        print(f"   Preço máximo: R$ {max(precos):.2f}")
        print(f"   Preço médio:  R$ {sum(precos)/len(precos):.2f}")
        com_desconto = [p for p in produtos if p.get("desconto_pct")]
        print(f"   Com desconto: {len(com_desconto)} ({len(com_desconto)/len(produtos)*100:.1f}%)")

asyncio.run(main())

Retry e Resiliência — Lidar com Falhas Graciosamente

python — utils/retry.py
import asyncio, functools
from typing import TypeVar, Callable, Any

T = TypeVar("T")

def retry_async(
    max_tentativas: int = 3,
    delay: float = 2.0,
    backoff: float = 2.0,
    excecoes: tuple = (Exception,)
):
    """Decorator de retry com backoff exponencial para funções async."""
    def decorator(func: Callable) -> Callable:
        @functools.wraps(func)
        async def wrapper(*args, **kwargs) -> Any:
            tentativa = 0
            delay_atual = delay

            while tentativa < max_tentativas:
                try:
                    return await func(*args, **kwargs)
                except excecoes as e:
                    tentativa += 1
                    if tentativa >= max_tentativas:
                        print(f"❌ {func.__name__} falhou após {max_tentativas} tentativas: {e}")
                        raise
                    print(f"⚠ {func.__name__} falhou (tentativa {tentativa}/{max_tentativas}): {e}")
                    print(f"  Aguardando {delay_atual:.1f}s antes de tentar novamente...")
                    await asyncio.sleep(delay_atual)
                    delay_atual *= backoff

        return wrapper
    return decorator

# Uso:
@retry_async(max_tentativas=3, delay=2.0, backoff=2.0)
async def extrair_pagina_resiliente(page):
    await page.wait_for_selector(".product-card", timeout=10_000)
    return await extrair_pagina_produtos(page)

⚠️ Armadilhas Comuns

💾 Salvar auth_state.json no git
O arquivo de estado de autenticação contém cookies e tokens reais. Adicione ao .gitignore: auth_state.json, *.cookies, session_*.json.
⚡ Velocidade sobre-humana
Clicar e preencher campos em milissegundos é um sinal claro de bot. Use slow_mo=100 em desenvolvimento e adicione await asyncio.sleep(random.uniform(0.5, 2.0)) entre ações críticas.
🔄 Não validar dados extraídos
Sem Pydantic, um campo ausente ou mal formatado pode silenciosamente salvar None ou string vazia no banco. Use sempre schemas com validators para falhar rápido e explicitamente.
📄 Processar páginas sem verificar paginação
Extrair apenas a primeira página e assumir que são todos os dados é um bug comum. Sempre verifique se há próxima página após extrair a atual, não antes.
🧠 Não fechar contextos e browsers
Cada browser.new_context() cria um processo. Sem fechar, acumulam-se processos Chromium zumbis que consomem memória. Use sempre context managers (async with).
